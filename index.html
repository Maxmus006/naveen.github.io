VAPT & Linux Commands (For authorized testing & learning only)

Legal / Ethical Notice: The commands below are shown for defensive learning, auditing, and authorized penetration tests only. Do NOT run them against systems you do not own or do not have explicit written permission to test.

Common VAPT / Recon Commands
# Basic network/service discovery (authorized target)
nmap -sS -Pn -sV -p 22,80,443 example.com

# Quick top ports scan
nmap -T4 --top-ports 100 example.com

# HTTP headers and quick check
curl -I https://example.com

# Directory brute-force (replace wordlist path)
gobuster dir -u https://example.com -w /usr/share/wordlists/dirb/common.txt -x php,html,txt

# Web server scan (non-destructive)
nikto -host https://example.com
Linux / Sysadmin & Forensics Helpers
# Update & upgrade packages
sudo apt update && sudo apt upgrade -y

# Check listening ports & services
ss -tuln

# Real-time logs (systemd)
sudo journalctl -f

# Monitor auth logs (login attempts)
sudo tail -f /var/log/auth.log

# Quick disk & permission checks
df -h
sudo find / -perm -4000 -type f 2>/dev/null

# Basic hardening (private key perms example)
chmod 600 ~/.ssh/id_rsa
chown $USER:$USER ~/.ssh/id_rsa
Simple Automation & Recon Snippet
# Example: combine a quick nmap + curl check for authorized target
TARGET="example.com"
nmap -Pn -sV -p 80,443 $TARGET -oN recon_${TARGET}_nmap.txt
curl -sI https://$TARGET | grep -i "server\|x-powered-by" > recon_${TARGET}_http.txt
echo "Saved recon_${TARGET}_*.txt"
